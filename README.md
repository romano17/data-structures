## About The Project

My java implementations of some of the most common data structures plus the solutions of problem solving challenges
found in interview preparation books/sites.

## Data Structures

* Array list.
* Hash table.
* Single linked list.
* Queue.
* Priority queue.
* Stack.
* Doubly linked list.
* Binary tree.
* AVL tree.
* Max heap.
* Prefix tree.
* Graph.
* Weighted graph.

* Java 11
* Gradle
